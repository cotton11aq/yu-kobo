# Yu-工房

手づくり雑貨ショップLP。

手づくりのポーチやかばんのECサイトにつながるLPを作成しました。  
オーダーメイド対応を強みにしており、「こんなポーチがあったらいいな」を
叶えるお手伝いをしたいという想いを込めて作成しました。
